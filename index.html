About me

I ponder that why not learn Python for a month and if don't find it interesting at all, then I pursue something else.

In July 2024, I decided to learn Python as I was not satisfied with my career as an accountant. I started with the book called Python Crash Course by Eric Mathes, which gave me a foundational knowledge about Python. Within three months, I gained a lot of momentum and started to enjoy the coding. I knew knowing Python is not enough so I decided to learn about machine learning. I research about the prerequisites to know about my knowledge gap.
